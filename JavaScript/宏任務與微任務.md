
### 宏任務

| #                     | Browser | Node |
| --------------------- | ------- | ---- |
| I/O                   | V       | V    |
| setTimeout            | V       | V    |
| setInterval           | V       | V    |
| requestAnimationFrame | V       | X    |

### 微任務

| #                          | Browser | Node |
| -------------------------- | ------- | ---- |
| process.nextTick           | X       | V    |
| MutationObserver           | V       | X    |
| Promise.then catch finally | V       | V    |
async/await 是基於 Promise 的封裝，所以效果類似 Promise.then


```js
setTimeout(() => alert("timeout"));

Promise.resolve()
	.then(() => alert("promise"));

alert("code");

// new Promise 執行為同步函數
// code -> promise -> timeout
```
