
## 工廠函數 factory function 

> 封裝創建物件的邏輯，返回新創建的對象。為避免使用 new 的方式。

### ex.

```JS
function createPerson(name, age) {
	return {
		name: name,
		age: age,
		sayHello: function() {
			console.log(`Hello, my name is ${this.name}`);
		}
	};
}

const person = createPerson('John', 30);

person.sayHello(); // 輸出：Hello, my name is John
```
